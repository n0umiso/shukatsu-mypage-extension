# 変更履歴

形式は [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に準拠。

## [Unreleased]

### Added
- ホーム画面（締切が近い・クイックログイン・登録数/選考中/今週の締切の集計）
- 選考管理ビュー（8ステージの進捗バー一覧＋ステージ別カンバンの切替）
- 選考ステージモデル `lib/stages.js`（気になる→…→内定の8段＋お祈り/辞退）
- ログイン動線の一体化（「ログイン」で対象サイトを開き ID/PW を自動入力、自動送信なし）
- i-web 学校選択ウィザード（2〜5ページ目）の自動入力（ラベル一致でラジオを選択）
- プロフィールに学校区分・大学所在地・文理を追加
- 最終アンケートの性別をラベル一致で汎用選択
- ⚡自動入力ボタンを学校ウィザード/アンケートページでも表示
- 高校名フィールドを追加し、質問文キーワードで自由記述を補完する汎用ルール（`textRules`）を実装（企業横断）
- 高校所在地フィールドを追加
- GitHub 公開リポジトリ作成

### Changed
- デザイン全面刷新：配色を案A（ネイビー×ウォームグレー、70/25/5・補色アクセント）に統一、ライトサイドバー化
- 絵文字を全廃し線アイコン（`lib/icons.js`）に置き換え。アバター色もくすんだ調和色に集約
- ツールバーのポップアップを刷新：現在ページの認識（ログイン/ID・PWコピー/取込）、締切が近い、クイックログインの起動特化UIに。編集・全管理はダッシュボードへ集約
- 情報・設定ページにサイドバーを追加し、各ビューと相互移動可能に（迷子バグ修正）
- 入力例プレースホルダから実在の固有名詞を排除（○○大学 等の伏せ字に）
- Entry の `status` を選考ステージ `stage` に刷新（スプレッドシート列も「選考ステージ」に）
- ダッシュボードのナビをホーム/マイページ/選考管理/締切/設定に再編

## [0.1.0] - 2026-06-13

### Added
- MV3 Chrome 拡張の基盤（background / content / popup）
- マイページの自動キャプチャ（ログイン検知で ID/PW/URL を保存）
- 締切の汎用抽出（日付 × キーワード）
- Google スプレッドシート同期（GAS Web アプリ、id キーで upsert）
- 全画面ダッシュボード（カード一覧・企業リンク・検索・ソート・締切カレンダー）
- 業界フィールドと業界順ソート/フィルタ
- 個人情報の自動入力（i-web 新規登録フォーム対応、`autofillIweb`）
- 情報入力・設定ページ（プロフィール + 同期/機能設定）
- ページ内に ⚡自動入力 / 📅締切抽出 ボタンを注入
