# プロジェクト進行管理

就活マイページ マネージャー（自作版）の開発ロードマップと進捗。
最終更新: 2026-06-14

## ゴール

就活マイページ（i-web / axol / e2r の3プラットフォーム）について、
**ログイン情報・締切の一元管理**＋**個人情報の自動入力**＋**スプレッドシート同期**を行う Chrome 拡張。
iOS / モバイルからも締切・ダッシュボードを閲覧可能にする（PWA）。
