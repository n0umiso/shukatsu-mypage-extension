Features

就活の「あのページどこだっけ」を解決

企業ごとにバラバラのマイページを一つのダッシュボードに集約。もうブックマークの山を掘り返す必要はありません。

ワンクリック自動取込

マイページにログインした状態でボタンを押すだけ。企業名・URL・ID・パスワード・締切を一発で記録します。

ダッシュボードで一覧管理

全企業をカードで表示。業界フィルタ・検索・ソートでサッと見つかります。ID/PW のコピーもワンクリック。

締切を自動抽出

ページ内の「ES」「面接」「説明会」等の締切日をキーワードから自動検出。7日以内の締切は赤く警告します。

スプレッドシート同期

Google スプレッドシートへワンクリックでバックアップ。PCが壊れてもデータは安全です。

選考ステージ管理

ES → 一次面接 → 最終面接 → 内定。進捗をカンバンボードと進捗バーで見える化します。

i-web 自動入力

プロフィールを一度登録すれば、i-web の新規登録フォームを自動で埋めてくれます。何度も住所を打つ必要なし。

Setup

はじめての Chrome 拡張機能でも大丈夫

3ステップで使い始められます。Chrome ウェブストアからのインストールではなく、GitHub からダウンロードして手動で入れます。

1

拡張機能をダウンロード

GitHub からファイルを PC にダウンロードします。

  1. GitHub のページを開く
  2. 緑色の 「<> Code」 ボタン → 「Download ZIP」 をクリック
  3. ダウンロードした ZIP を展開（右クリック →「すべて展開」）
2

Chrome に読み込む

「デベロッパーモード」で拡張機能を手動インストールします。初めてでも1分で終わります。

  1. Chrome のアドレスバーに chrome://extensions と入力して Enter
  2. 右上の 「デベロッパーモード」 をオンにする
  3. 「パッケージ化されていない拡張機能を読み込む」 をクリック
  4. 先ほど展開したフォルダを選択 → OK

Chrome の右上にアイコンが出れば成功です。ピン留めしておくと便利です。

3

マイページにログインして取り込む

いつも通りマイページにログインし、ポップアップの「今のページを保存」を押すだけ。

  1. 企業のマイページにログインする
  2. Chrome 右上の拡張機能アイコンをクリック
  3. 「今のページを保存」 を押す
  4. 企業名・URL・ID・パスワード・締切が自動で記録されます

スプレッドシート同期も使いたい？（任意）

データのバックアップや iPhone からの閲覧には、Google スプレッドシートとの同期が便利です。 README の手順に沿って GAS Web アプリをデプロイし、拡張機能の設定画面で URL を登録してください。

スプレッドシート同期を使わなくても、拡張機能単体で問題なく動作します。

Security

あなたのデータは、あなたの端末に

ログイン情報は外部サーバーに送信されません。すべてのデータは Chrome のローカルストレージに保存されます。

ローカル保存

ID・パスワードは chrome.storage.local に保存され、端末から出ません。

同期は任意

スプレッドシート同期は任意機能です。パスワードの送信も設定でオフにできます。

トークン認証

同期時にトークン認証を設定でき、URL を知っているだけではアクセスできないようにできます。

オープンソース

コードはすべて GitHub で公開。何をしているか誰でも確認できます。

FAQ

よくある質問

Chrome ウェブストアにありますか？
現在は GitHub からの手動インストールのみです。デベロッパーモードで読み込めば、ウェブストア版と同じように動作します。
Edge や Brave でも使えますか？
はい。Chromium ベースのブラウザ（Edge, Brave, Arc, Vivaldi 等）であれば同じ手順でインストールできます。
i-web 以外のマイページにも対応していますか？
ログイン情報の取込みと締切抽出はすべてのサイトで動作します。プロフィール自動入力は現在 i-web のみ対応しています。
パスワードは安全ですか？
パスワードは Chrome のローカルストレージに保存され、外部に送信されません。スプレッドシート同期時にパスワードを含めるかは設定で選べます（デフォルト ON）。気になる方はオフにしてください。
iPhone や Android から見られますか？
スプレッドシート同期を設定すれば、PWA（モバイル用Webアプリ）で企業一覧と締切を確認できます。編集は PC の拡張機能で行ってください。
アンインストールするとデータは消えますか？
はい、拡張機能を削除するとローカルのデータは消えます。スプレッドシートに同期していれば、そちらにバックアップが残ります。アンインストール前にダッシュボードの「エクスポート」でJSON形式のバックアップも取れます。
