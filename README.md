# 就活マイページ マネージャー

就活のマイページ（i-web 等）の **ログインID・パスワード・締切** を拡張機能内で一元管理し、
裏で **Google スプレッドシートにも自動同期** する Chrome 拡張機能です。

- 普段は拡張機能のポップアップで管理
- ログインフォーム送信を検知して ID/PW を自動キャプチャ
- ページ内の締切らしき日付（ES・説明会・面接など）を自動抽出
- 全データを GAS Web アプリ経由でスプレッドシートへ upsert（バックアップ）

---

## セットアップ

### 1. スプレッドシート側（GAS Web アプリ）

1. 同期先にしたい Google スプレッドシートを開く
2. メニュー **拡張機能 → Apps Script** を開く
3. `gas/Code.gs` の中身を貼り付けて保存
4. 右上 **デプロイ → 新しいデプロイ**
   - 種類: **ウェブアプリ**
   - 次のユーザーとして実行: **自分**
   - アクセスできるユーザー: **全員**（※URL を知っている人のみアクセス可。推測困難な URL）
5. 発行された **ウェブアプリ URL（`.../exec`）** をコピー

> 「全員」が不安な場合は「自分のみ」でも動きますが、拡張機能からの匿名 POST が拒否されます。
> 個人利用なら URL を秘匿する前提で「全員」が手軽です。

### 2. 拡張機能側

1. Chrome で `chrome://extensions` を開く
2. 右上の **デベロッパーモード** を ON
3. **パッケージ化されていない拡張機能を読み込む** → このフォルダを選択
4. 拡張機能の **詳細 → 拡張機能のオプション**（またはポップアップの ⚙）を開く
5. **GAS Web アプリ URL** を貼り付けて保存

## 使い方

### ポップアップ（クイック操作）

| 操作 | 説明 |
|------|------|
| マイページにログイン | フォーム送信／ログインボタン押下を検知し ID/PW・URL・締切を自動取り込み |
| 📋 一覧 | 全画面ダッシュボードをタブで開く |
| ＋ 今のページ | 表示中のページを手動で取り込む |
| ID / PW ボタン | クリップボードへコピー |
| ⟳ 同期 | 全データをスプレッドシートへ手動同期 |

### ダッシュボード（タブ表示）

`dashboard.html` を新規タブで開く全画面の一覧画面。

- 企業ごとに**カード表示**。企業名／URL リンクから各マイページへ直接ジャンプ
- **マイページを追加**（手動登録）
- **業界**をカード上でその場で変更、**業界順／更新順／企業名順**ソート・検索・業界フィルタ
- ID/PW のコピー・表示、締切バッジ（7日以内は赤）
- **締切カレンダー**タブ：全企業の締切を日付順に一覧

締切は `種別:日付`（例 `ES:2026/06/30`）の形式で 1 行 1 件。あと7日以内は赤く表示されます。

---

### 個人情報の自動入力（i-web 対応）

1. 拡張機能の **オプション（情報入力・設定）** を開き、基本情報・連絡先・住所・学歴を登録して保存
2. i-web の新規登録フォームを開くと、画面右下に **⚡自動入力 / 📅締切抽出** ボタンが出る
3. **⚡自動入力** を押すと、登録済みプロフィールがフォームへ一括入力される

- 対応マッピングは `content.js` の `autofillIweb()`。都道府県は名称→i-webコード（`01`〜`47`）へ自動変換。
- 「現住所と同じ」をオンにすると帰省先へ現住所をコピー。
- 他サイト（axol / e2r 等）は、フォームの `name` 一覧をもらえれば `AUTOFILLERS` に追加して対応できます。

## ⚠️ セキュリティについて

- ログイン情報は `chrome.storage.local`（端末内）に保存されます。
- **パスワードをスプレッドシートに送ると Google Drive 上に平文で残ります。**
  設定画面の「パスワードもスプレッドシートへ送信する」を **OFF** にすれば、
  パスワードは端末内にのみ保持され、シートには送られません。
- **同期トークン** を設定すると、GAS への読み書きにトークンが必要になります。
  1. GAS エディタ → プロジェクトの設定 → スクリプトプロパティで `SYNC_TOKEN` に任意の文字列を設定
  2. 拡張機能の「同期・機能設定」タブで同じ文字列を入力
  3. PWA にも同じ値を入力
- 共有スプレッドシートは使わないでください。
- ログイン情報の取り込みは自動では行われません。ポップアップの「今のページを保存」ボタンで明示的に取り込みます。

## iOS / モバイルで見る（PWA）

`pwa/` ディレクトリに、スプレッドシートのデータを **読み取り専用** で表示する PWA が入っています。

### セットアップ

1. `pwa/` を GitHub Pages や任意の静的ホスティングにデプロイ
2. iPhone の Safari で PWA の URL を開く
3. GAS Web アプリ URL と同期トークンを入力して「接続」
4. 「共有」→「ホーム画面に追加」でアプリ風に使える

締切・ダッシュボード・企業一覧を確認できます（編集は PC の拡張機能で行ってください）。

---

## 対応サイトの追加（締切抽出の精度向上）

`src/content.js` の `DEADLINE_KEYWORDS` / `guessType()` を調整すると抽出精度を上げられます。
特定サイト専用の抽出が必要になったら、`location.hostname` で分岐するロジックを追加してください。

## ファイル構成

```
manifest.json        … MV3 マニフェスト
src/
  background.js      … データ更新の単一窓口 + 同期トリガ
  content.js         … 手動キャプチャ・締切抽出・自動入力
  popup.html/js/css  … ポップアップ（クイック操作）
  dashboard.html/js  … 全画面ダッシュボード
  profile.html/js    … プロフィール・設定
  lib/store.js       … ローカル保存ヘルパー
  lib/sync.js        … GAS への同期
  lib/utils.js       … 共通ユーティリティ
  lib/stages.js      … 選考ステージ定義
  lib/icons.js       … SVG アイコン
gas/Code.gs          … スプレッドシート側 Web アプリ
pwa/                 … iOS / モバイル用 PWA（読み取り専用）
```
